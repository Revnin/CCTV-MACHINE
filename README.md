# CCTV-MACHINE
A black box penetration test on HackTheBox's CCTV machine achieving full root compromise via four vulnerabilities: default credentials, SQL injection (CVE-2024-51482), password hash cracking, and Remote Code Execution in motionEye (CVE-2025-60787)

📋 Project Overview
This repository contains the full penetration test report for the HackTheBox CCTV machine. The assessment simulated a real-world black box attack against a fictional CCTV security company called SecureVision, which was running two vulnerable camera management systems — ZoneMinder and motionEye.
Starting with only an IP address and no prior knowledge, a complete system compromise was achieved — gaining root access through a chain of four vulnerabilities.
